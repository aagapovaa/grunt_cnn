# grunt_cnn
Вариант свёрточной нейронной сети для автоматизированного дешифрирования сети грунтовых дорог. Код написан в рамках курсовой работы на тему "Автоматизированное дешифрирование сети грунтовых дорог по данным космической съемки сверхвысокого пространственного разрешения", выполненной Агаповой Е.Р. (кафедра картографии и геоинформатики МГУ)
